WissKI RelFinder
================

Integrates the RelFinder tool into WissKI.

This module provides another way of browsing your data in WissKI.
It makes use of the RelFinder visualisation library
(<http://www.visualdataweb.org/relfinder.php>).

For installation instructions see INSTALLATION.txt.

After enabling the module, the default behaviour of the RelFinder can be
configured at Site Configuration -> WissKI module settings -> RelFinder.

The relfinder can be launched via wisski/relfinder/relfinder.swf.
For your convenience you can add a button beneath the standard "Create",
"Navigate" and "Find" buttons by editing the "Secondary links" menu
(Site Buildung -> Menus -> Secondary links).
