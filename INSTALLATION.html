This module requires the RelFinder library installed in the libraries directory.

Install the library with the following steps:

  1. In your Drupal base directory, create the directory path sites/all/libraries/relfinder if not already there.
  2. Download the RelFinder library (to swf file) from here.
  3. Place the file in sites/all/libraries/relfinder as RelFinder.swf.
  4. Check on this page, if the warning disappears. If so, the library is installed correctly.

NOTE: In order to access the RelFinder a user must also have read permissions on the endpoint; at least wisski endpoint supports select.
